# Neural Shift - Metrics & Analytics Guide

## What to Track

### Traffic Metrics
- Daily unique visitors
- Session duration
- Pages per session
- Bounce rate
- Traffic source (direct, social, referral)

### User Behavior
- Programs viewed
- Programs started
- Session duration per program
- Return visitors
- Free → Pro conversion rate

### Revenue Metrics
- Upgrade attempts
- Successful purchases
- Revenue per day/week/month
- Refund rate
- Lifetime value (LTV)

### Funnel Stages
1. Landing page → Program selection
2. Program selection → Initialize
3. Initialize → Session complete
4. Session complete → Pricing view
5. Pricing view → Upgrade

---

## Where to Track

### Google Analytics 4
- Install GA4 on website
- Create events for: signup, upgrade, purchase
- Set up conversion funnels
- Link to Search Console

### Vercel Analytics
- Built-in, free
- Shows: visitors, views, performance
- Real-time dashboard

### Stripe Dashboard
- Revenue metrics
- Failed vs successful payments
- Refund tracking
- Customer lifetime value
