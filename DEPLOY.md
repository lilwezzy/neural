# Neural Engine Deployment Guide

To get your engine live and on your phone for SpiritFest:

### Step 1: Create the Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it `neural-engine`
3. **CRITICAL**: Set it to **Private** (to protect your manifestation engine).
4. Click "Create repository".

### Step 2: Push the Code
Copy and run these commands in your terminal:
```bash
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/neural-engine.git
git branch -M main
git push -u origin main
```

### Step 3: Go Live on Vercel
1. Go to [Vercel.com](https://vercel.com) and log in with GitHub.
2. Click **"Add New"** -> **"Project"**.
3. Select your `neural-engine` repo.
4. Click **"Deploy"**.
